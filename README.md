# Simple MCP Teaching Demos

Each folder is independent and focuses on one concept:

1. `01_mcp_server_basics` - build a server with tools, resources, and prompts.
   Open [`01_mcp_server_basics/exercise.html`](01_mcp_server_basics/exercise.html).
2. `02_mcp_inspector` - inspect and manually test every MCP primitive.
   Open [`02_mcp_inspector/exercise.html`](02_mcp_inspector/exercise.html).
3. `03_azure_agent_with_mcp` - give an Azure OpenAI model a remote MCP server.
   Open [`03_azure_agent_with_mcp/exercise.html`](03_azure_agent_with_mcp/exercise.html).
4. `04_mcp_apps_widgets` - render weather, time, and calendar widgets.
   Open [`04_mcp_apps_widgets/exercise.html`](04_mcp_apps_widgets/exercise.html).
5. `05_huggingface_transformers_pipelines` - explore common Transformer
   pipelines in Google Colab.
   Open [`05_huggingface_transformers_pipelines/exercise.html`](05_huggingface_transformers_pipelines/exercise.html).
6. `06_behind_transformers_pipeline` - inspect tokenization, hidden states,
   logits, softmax, and labels behind a sentiment pipeline.
   Open [`06_behind_transformers_pipeline/exercise.html`](06_behind_transformers_pipeline/exercise.html)
   and upload [`06_behind_transformers_pipeline/behind_transformers_pipeline.ipynb`](06_behind_transformers_pipeline/behind_transformers_pipeline.ipynb)
   to Google Colab.
7. `07_simple_rag_demo` - build a simple RAG system over PEFT GitHub issues with BGE,
   FAISS, a quantized Zephyr model, and LangChain.
   Open [`07_simple_rag_demo/exercise.html`](07_simple_rag_demo/exercise.html)
   and upload [`07_simple_rag_demo/simple_rag_demo.ipynb`](07_simple_rag_demo/simple_rag_demo.ipynb)
   to Google Colab.
8. `08_advanced_rag_demo` - improve RAG with token-aware Markdown chunking,
   diagnostics, cosine retrieval, embedding visualization, ColBERT reranking,
   source tracking, and evaluation.
   Open [`08_advanced_rag_demo/exercise.html`](08_advanced_rag_demo/exercise.html)
   and upload [`08_advanced_rag_demo/advanced_rag_demo.ipynb`](08_advanced_rag_demo/advanced_rag_demo.ipynb)
   to Google Colab.

These are intentionally small classroom demos, not production templates.
